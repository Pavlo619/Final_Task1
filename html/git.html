Git - всему голова!

Инициализация репозитория:

  • GIT INIT - Инициализирует новый репозиторий Git в текущей директории.

Создание и управление ветками:

  • GIT BRANCH: Создает, перечисляет или удаляет ветки.
  • GIT CHECKOUT: Переключается на другую ветку.
  • GIT MERGE: Объединяет ветку с текущей веткой.

Изменение файлов и добавление в репозиторий:

  • GIT STATUS: Показывает состояние репозитория (какие изменения были внесены, что не отслеживается).
  • GIT ADD: Добавляет изменения в индекс (область подготовки к фиксации).
  • GIT COMMIT: Сохраняет изменения в репозитории.

Работа с удаленными репозиториями:

  • GIT REMOTE: Добавляет, удаляет или перечисляет удаленные репозитории.
  • GIT PUSH: Отправляет изменения из локального репозитория в удаленный.
  • GIT PULL: Получает изменения из удаленного репозитория и объединяет их с локальным.

Дополнительные команды:

  • GIT CLONE: Создает копию удаленного репозитория.
  • GIT DIFF: Показывает изменения между двумя коммитами или файлами.
  • GIT TAG: Создает метку для определенного коммита.
